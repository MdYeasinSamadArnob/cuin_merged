## 🐛 **Common Issues & Solutions**

### **Issue 1: FileNotFoundError: blocking_analysis.csv**
**Cause:** Blocking stage skipped or failed
**Solution:** Check logs for blocking errors, ensure Spark has write permissions

### **Issue 2: OutOfMemoryError during scoring**
**Cause:** Not filtering data to candidates only
**Solution:** Fixed version filters data before scoring (line 245-280)

### **Issue 3: Spark session creation fails**
**Cause:** Missing HADOOP_HOME on Windows
**Solution:** Run `python setup_hadoop_windows.py` or install winutils.exe

### **Issue 4: Empty results**
**Cause:** Threshold too high (0.95) or no actual duplicates
**Solution:** Lower threshold to 0.85 for testing
